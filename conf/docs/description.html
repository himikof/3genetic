3Genetic

Эта программа представляет собой виртуальную лабораторию для разработки генетических алгоритмов и их изучения. Функциональность приложения обеспечивается за счет подключения к ядру модулей - плагинов.

Ядро лаборатории поддерживает интерфейс пользователя, включая вывод графиков, а также связывает плагины для формирования исследуемого алгоритма.

Лаборатория используется следующим образом.

  1. Формулируется решаемая задача в виде текстового описания задачи и выбранной функции приспособленности (фитнесс-функции), которая задается математически. Формулировка задачи оформляется в виде плагина.
  2. Для решаемой задачи выбирается структура особи (например, конечные автоматы с полными таблицами переходов, конечные автоматы с сокращенными таблицами переходов, система автоматов, нейронные сети, и т. д.). На этом этапе формируется плагин "особь".
  3. Формируется конфигурация особи, состоящая из конфигурационных параметров, например, число состояний автомата, а также из операторов скрещивания (кроссовера) и мутации. Каждый опертор оформляется в виде плагина.
  4. Выбирается схема алгоритма, включая оператор отбора. Осуществляется настройка конфигурационных параметров алгортма, таких как, например, размер поколения, вероятность мутации, вероятность скрещивания и т. д. Существуют различные типы алгоритмов, такие как, например, канонический - последоватеное применение операторов отбора, скрещивания и мутации, островной, клеточный. Схема алгоритма оформляется в виде плагина. Операторы отбора также оформляются в виде плагинов.
  5. Осуществляется запуск алгоритма.
  6. В результате работы алгоритма пользователю представляются графики, например, график зависимости максимальной функциии приспособленности от номера поколения. Пользователь может добавить те функции, которые ему необходимы. Новые функции оформляются в виде плагинов.
  7. После постановки алгоритма на паузу, доступен простмотр визуализатора любой особи из текущего поколения, а также лучших особей из каждого поколения. Визуализаторы оформляются в виде плагинов.

Для использования 3Genetic от пользователя требуется наличие навыков программирования на языке Java. Также предполагается знание основных схем генетических алгоритмов.

Модули, входящие в базовый комплект

Ядро

Ядро программы включает в себя три модуля.
  1. common.jar — содержит все необходимые для разработки плагинов интерфейсы.
  2. core.jar — ядро системы и пользовательский интерфейс.
  3. util.jar — содержит реализацию вспомогательных классов.

Плагины

В базовый комплект входят следующие плагины:
  1. algorithms/simple.jar — конанический генетический алгоритм.
  2. functors/max.jar — плагин для отображения графика фунции максимальной функции приспособленности от номера поколения.
  3. functors/mean.jar — плагин для отображения графика фунции максимальной функции приспособленности от номера поколения.
  4. tasks/ant.jar — формулировка задачи об "Умном муравье".
  5. individuals/IndividualG — тестовая особь.
  6. individuals/mealy — особь, представляющая собой конечный автомат Мили, для задачи об "Умном муравье".
  7. visualizators/sAnt.jar — визуализатор для особи mealy.
