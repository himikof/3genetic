Система конечных автоматов Мили

Данная особь представляет из себя управляющую систему для умного муравья в виде системы из двух автоматов Мили, взаимодействующих за счет вложенности. Генетические операторы определены покомпонентно для внешнего и вложенного автомата.

Подробнее об этой особи можно прочитать в следующей статье: http://is.ifmo.ru/genalg/_scm2008_sokolov.pdf
